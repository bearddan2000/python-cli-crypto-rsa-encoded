# python-cli-crypto-rsa-encoded

## Description
Compare hashed passwords. RSA
is an asymmetric algorithm that
uses a 3072 bit key. Encryption is
done by public key while decryption
is by private key.

## Tech stack
- python 3.8

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
